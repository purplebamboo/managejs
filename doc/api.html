transfer

用于将一段javascript字符串 解析成一个jsnodelist对象.

方法 exports.transfer() exports transfer
示例 undefined undefined undefined
参数 jsString(string) 需要解析的字符串
参数 options(object) 配置项
参数 options.remainSpace(boolean) 是否保留各种空格，为true的话会保留以前的格式。会消耗性能，并且容易出错
参数 options.remainComment(boolean) 是否保留注释
返回 JsNodeList 根节点，JsNodeList对象，可以使用提供的一系列方法
